LifeTrack – Gestor de Vida Personal

Una aplicación web que mezcla:

✔ Finanzas
✔ Proyectos personales
✔ Aprendizaje / entretenimiento

Todo en un solo lugar.

¿Qué sería?

LifeTrack sería como un panel personal donde puedes gestionar:

Tu dinero 

Tus metas 

Una especie de dashboard de crecimiento personal.

 Funcionalidades iniciales
1. Mini Finanzas Personales

Agregar ingresos

Agregar gastos

Ver saldo total

Con colores:
🟢 Verde = vas bien
🔴 Rojo = estás gastando más

2. Que tenga buscador Inteligente

3. Sistema de Tags

Ejemplo:

Productividad

Aprendizaje

Diversión

Inversión

Filtrado rápido.

 Futuro 

Gráficas de gastos 

Login de usuarios
